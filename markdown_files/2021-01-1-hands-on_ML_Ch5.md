# 서포트 벡터 머신

- 복잡한 분류 문제, 작거나 중간 크기의 데이터셋에 적합하다
- 특성의 스케일에 민감하다

## 선형 SVM 분류

**SVM 분류기**
- 클래스 사이에 가장 폭이 넓은 도로를 찾는 것 **(라지 마진 분류)**


**서포트 벡터**
- 도로 바깥쪽에 훈련 샘플을 더 추가해도 결정 경계에는 전혀 영향을 미치지 않는다.
- 도로 경계에 위치한 샘플에 의해 전적으로 결정된다.
